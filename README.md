# AI Language Tutor

Interactive language learning platform using Xiaomi MiMo for conversation practice

## 🎯 Features

- Powered by Xiaomi MiMo API
- Simple and intuitive interface
- Real-time processing
- Token consumption: 500K tokens/day

## 🚀 Quick Start

```bash
git clone https://github.com/mail-eth/ai-language-tutor.git
cd ai-language-tutor
npm install
cp .env.example .env
# Add your MIMO_API_KEY
npm start
```

## 📊 Use Case

This project demonstrates practical AI integration using Xiaomi MiMo API for real-world applications.

## 🛠️ Tech Stack

- Xiaomi MiMo API
- Node.js
- Express
- Simple HTML/CSS frontend
